My workspace is Arch Linux with KDE Plasma. It's where I feel most at home — the terminal, the tiling, the customization, the control. So when I set out to build a portfolio, I didn't want some generic template. I wanted it to feel like *my* machine. Boot sequence, lock screen, desktop, windows — the whole experience. Because this is what my everyday looks like, and I figured that's the most honest way to show who I am and what I do.

Welcome to my desktop.

---

Built with **React 19**, **Vite 8**, **Tailwind CSS v4**, **React Router DOM v7**, and **JetBrains Mono**.
